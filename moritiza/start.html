Installation

Server Requirements

Make sure your server meets the following requirements:

  • PHP >= 7.2.0
  • Ctype PHP Extension
  • JSON PHP Extension
  • Mbstring PHP Extension
  • Curl PHP Extension
  • PDO PHP Extension

MoriTiza utilizes Composer to manage its dependencies. So, make sure you have Composer installed on your machine.

Via Composer Create-Project

composer create-project mortezanasiri/moritiza
